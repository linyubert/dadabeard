# 節奏幫爸爸拔鬍子

一款網頁版節奏小遊戲。玩家需要先看示範：爸爸會依照小鼓節奏長出鬍子；接著輪到玩家照同樣節奏按鍵，把鬍子拔掉。

## 遊戲特色

- 首頁滿版背景圖
- Start 後進入規則說明頁
- 首頁與規則頁可播放 BGM
- 正式開始遊戲後 BGM 會停止
- 出題階段：小鼓聲提示節奏，爸爸長出鬍子
- 答題階段：玩家照節奏拔鬍子
- 支援普通按鍵與長按節奏
- 支援鍵盤、滑鼠與觸控操作
- 結算畫面與再玩一次功能
- 自動符合瀏覽器視窗大小

## 操作方式

| 操作 | 說明 |
|---|---|
| `Space` | 拔鬍子 / 長按鬍子 |
| 滑鼠點擊 | 拔鬍子 / 長按鬍子 |
| 觸控 | 拔鬍子 / 長按鬍子 |
| `F11` | 建議進入全螢幕，取得較佳體驗 |

## 遊戲流程

1. 進入首頁
2. 點擊 `Start`
3. 閱讀規則說明
4. 點擊「我懂了，開始遊戲」
5. 聽小鼓聲，看爸爸長鬍子
6. 輪到玩家時，照節奏拔鬍子
7. 完成後顯示結算畫面
